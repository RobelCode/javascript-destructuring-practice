# JavaScript Destructuring Practice

Welcome to the **Destructuring Exercise**! This project helps you learn and practice modern JavaScript features like object and array destructuring, default values, nested destructuring, and variable swapping through fun, story-driven exercises.

## Features

- Extract values from objects and arrays using destructuring.
- Assign default values to ensure code robustness.
- Rename variables while destructuring objects.
- Work with nested data structures.
- Swap variables efficiently without temporary variables.
- Use destructuring in function parameters.

## Exercises Included

1. **Coordinates Extraction** – Unveiling hidden x and y coordinates.
2. **Key Locations** – Isolating important map locations.
3. **Mysterious Door** – Using default values for missing properties.
4. **Riddle Translation** – Renaming destructured properties.
5. **Array of Elements** – Extracting essential elements from an array.
6. **Skipping Stones** – Selective array extraction.
7. **Array of Shadows** – Using rest operator in arrays.
8. **Decoding Directions** – Destructuring in function parameters.
9. **Potion of Clarity** – Handling optional properties with defaults.
10. **Casting Spells** – Array destructuring in functions.
11. **Nested Secret** – Extracting deeply nested values.
12. **Swapping Stones** – Swapping variables with array destructuring.
